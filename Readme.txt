# Assignment README

This zip folder contains solutions to the following tasks in Java, JavaScript, and HTML:

## Java
### Task A
- Create an array with the values (1, 2, 3, 4, 5, 6, 7) and shuffle it.

### Task B
- Enter a Roman Number as input and convert it to an integer. (e.g., IX = 9)

### Task C
- Check if the input is a pangram or not. (A pangram is a sentence that contains all the alphabet from a to z).

## JavaScript
### Task A
- Take a sentence as input and reverse every word in that sentence.

### Task B
- Perform sorting of an array in descending order.

## HTML
### Task A
- Create a basic calculator using HTML, CSS, and JavaScript with the functionality of addition, subtraction, multiplication, and division.

### Task B
- Create a survey form with fields; First Name, Last Name, Date of Birth, Country (dropdown), Gender (checkbox), Profession, email, and mobile number. All input fields are necessary to submit the form. Create two buttons, Submit and Reset. Reset will reset the form, and upon clicking Submit, it will validate the fields and display the selected values in a popup. On closing the popup, the form should reset all values.

Please review the respective code files and resources provided for each task within the assignment folder for detailed implementations.
